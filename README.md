# MemB-Chat
Real-time chat application, created using ReactJs, Redux, ExpressJs, MongoDB and Socket.io

### Try it yourself: 
[MemB Chat](https://ayanisation.github.io/MemB-Chat/)

## Setup:

1. cd client
2. npm install
3. npm start
